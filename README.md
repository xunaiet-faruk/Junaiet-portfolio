## 📖 About This Project

This is my personal portfolio website showcasing my skills, projects, and experience as a **Full Stack Developer**. The portfolio features a stunning 3D animated background, responsive design, smooth animations, and interactive project showcases.

### 🎯 Key Features

- ✨ **3D Animated Background** - Interactive Three.js scene with orbiting elements
- 🎨 **Modern UI/UX** - Glassmorphism design with gradient effects
- 📱 **Fully Responsive** - Perfect on all devices (mobile, tablet, desktop)
- 🚀 **Smooth Animations** - Framer Motion powered transitions
- 💼 **Project Showcase** - Interactive project cards with full details
- 📧 **Contact Form** - EmailJS integration for direct messaging
- 🎭 **Skill Section** - Hexagonal skill cards with animated borders
- 🌙 **Scroll Progress Indicator** - Visual scroll tracking
